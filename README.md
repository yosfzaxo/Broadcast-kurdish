# Discord Broadcast Bot

A simple Discord bot for broadcasting messages to server members based on their online/offline status.

## Zanyari Boti

- Broadcast BO HAMI KASA.
- Broadcast BO YET ONLINE.
- Broadcast BO YET OFFLINE.
